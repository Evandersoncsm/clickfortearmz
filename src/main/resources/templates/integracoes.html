Como consultar a API Omie

As consultas da API Omie usam o método HTTP POST, inclusive para listar ou consultar dados. Cada serviço possui seu próprio endpoint, informado na documentação oficial da Omie.

Exemplo: listar produtos

POST https://app.omie.com.br/api/v1/geral/produtos/
Content-Type: application/json

{
  "call": "ListarProdutos",
  "app_key": "SUA_APP_KEY",
  "app_secret": "SEU_APP_SECRET",
  "param": [
    {
      "pagina": 1,
      "registros_por_pagina": 50
    }
  ]
}
O App Secret deve ser usado somente pelo servidor. Nunca coloque essa credencial em JavaScript, HTML, Swagger público, logs ou mensagens de erro.
